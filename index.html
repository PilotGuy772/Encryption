Intelligence Encrypter

V 1.0

This is a simple encryption platform that can encrypt any plaintext message (no numbers) into another, encrypted, plaintext message. All you need is a five number code. Click here to learn more

Encrypt or Decrypt!

  • First, click 'Enter Code' below, and enter five numbers of any positive value seperated by spaces.
  • Then, click 'Enter Message', and paste in your plaintext (no numbers) message
  • Click 'Encrypt' and copy the resulting encrypted message.
    • When sending the message, make sure the recipient also has the five-number-code for decryption. Note that if you are encrypting, you can leave the code box blank for a random code.
